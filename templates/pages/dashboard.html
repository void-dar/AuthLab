Token is stored in HttpOnly cookie — not accessible via JavaScript (XSS protection ✓)
Why HttpOnly? JavaScript cannot read HttpOnly cookies, protecting your token from XSS attacks. This is a critical security best practice — even if an attacker injects malicious JS, they cannot steal your session token.
