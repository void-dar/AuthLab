What is MFA?

Multi-Factor Authentication requires two or more of: something you know (password), something you have (phone/device), something you are (biometrics). TOTP (Time-based One-Time Password) uses a shared secret + current time to generate a 6-digit code that changes every 30 seconds.

⚙️ How TOTP Works (RFC 6238)

1️⃣
Shared Secret — During setup, server and authenticator app exchange a secret key (usually via QR code)
↓
2️⃣
Time Counter — Both sides compute T = floor(Unix_time / 30) — a new number every 30 seconds
↓
3️⃣
HMAC-SHA1 — Compute HMAC-SHA1(secret, T) — a cryptographic fingerprint
↓
4️⃣
Truncation — Extract 6 digits from the HMAC output using dynamic truncation
↓
✅
Verify — Server computes the same value. If codes match, authentication succeeds (±1 window for clock drift)
